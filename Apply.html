Job Description

As a Web Developer at Web Hub, you will be responsible for creating stunning, responsive, and functional websites. You will work with our design and backend teams to deliver high-quality web solutions for a wide range of clients.

Your main responsibilities include:

  • Developing web pages and applications using HTML, CSS, and JavaScript
  • Collaborating with designers to implement attractive user interfaces
  • Optimizing websites for maximum performance and scalability
  • Maintaining and improving existing websites

Job Specifications

Responsive Design

Develop websites that look great on any device.

Collaborative Work

Work with designers and backend developers on various projects.

SEO Optimization

Ensure websites are optimized for search engines.

Required Qualifications

  • HTML & CSS: Proficiency in HTML and CSS for building responsive layouts.
  • JavaScript: Experience with JS for interactive and dynamic websites.
  • Git: Knowledge of Git for version control and collaboration.
  • Teamwork: Ability to work in a team and communicate effectively.
